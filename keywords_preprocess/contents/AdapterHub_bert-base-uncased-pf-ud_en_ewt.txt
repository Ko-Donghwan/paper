<span>
		Adapter <code>AdapterHub/bert-base-uncased-pf-ud_en_ewt</code> for bert-base-uncased
	</span>
</h1>
<p>An <a href="https://adapterhub.ml" rel="nofollow">adapter</a> for the <code>bert-base-uncased</code> model that was trained on the <a href="https://adapterhub.ml/explore/dp/ud_ewt/" rel="nofollow">dp/ud_ewt</a> dataset and includes a prediction head for dependency parsing.</p>
<p>This adapter was created for usage with the <strong><a href="https://github.com/Adapter-Hub/adapter-transformers" rel="nofollow">adapter-transformers</a></strong> library.</p>

<span>
		Usage
	</span>
</h2>
<p>First, install <code>adapter-transformers</code>:</p>
<pre><code>pip install -U adapter-transformers
</code></pre>
<p><em>Note: adapter-transformers is a fork of transformers that acts as a drop-in replacement with adapter support. <a href="https://docs.adapterhub.ml/installation.html" rel="nofollow">More</a></em></p>
<p>Now, the adapter can be loaded and activated like this:</p>
<pre><code class="language-python"><span class="hljs-keyword">from</span> transformers <span class="hljs-keyword">import</span> AutoModelWithHeads

model = AutoModelWithHeads.from_pretrained(<span class="hljs-string">"bert-base-uncased"</span>)
adapter_name = model.load_adapter(<span class="hljs-string">"AdapterHub/bert-base-uncased-pf-ud_en_ewt"</span>, source=<span class="hljs-string">"hf"</span>, set_active=<span class="hljs-literal">True</span>)
</code></pre>
