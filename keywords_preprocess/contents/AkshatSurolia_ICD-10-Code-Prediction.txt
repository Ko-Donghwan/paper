<span>
		How to use the model
	</span>
</h2>
<p>Load the model via the transformers library:</p>
<pre><code>from transformers import AutoTokenizer, BertForSequenceClassification
tokenizer = AutoTokenizer.from_pretrained("AkshatSurolia/ICD-10-Code-Prediction")
model = BertForSequenceClassification.from_pretrained("AkshatSurolia/ICD-10-Code-Prediction")
config = model.config
</code></pre>
<p>Run the model with clinical diagonosis text:</p>
<pre><code>text = "subarachnoid hemorrhage scalp laceration service: surgery major surgical or invasive"
encoded_input = tokenizer(text, return_tensors='pt')
output = model(**encoded_input)
</code></pre>
<p>Return the Top-5 predicted ICD-10 codes:</p>
<pre><code>results = output.logits.detach().cpu().numpy()[0].argsort()[::-1][:5]
return [ config.id2label[ids] for ids in results]
</code></pre>
