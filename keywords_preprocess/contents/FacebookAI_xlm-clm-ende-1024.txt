<span>
		Model Details
	</span>
</h1>
<p>The XLM model was proposed in <a href="https://arxiv.org/abs/1901.07291" rel="nofollow">Cross-lingual Language Model Pretraining</a> by Guillaume Lample, Alexis Conneau. xlm-clm-ende-1024 is a transformer pretrained using a causal language modeling (CLM) objective (next token prediction) for English-German.</p>

<pre><code class="language-python"><span class="hljs-keyword">import</span> torch
<span class="hljs-keyword">from</span> transformers <span class="hljs-keyword">import</span> XLMTokenizer, XLMWithLMHeadModel

tokenizer = XLMTokenizer.from_pretrained(<span class="hljs-string">"xlm-clm-ende-1024"</span>)
model = XLMWithLMHeadModel.from_pretrained(<span class="hljs-string">"xlm-clm-ende-1024"</span>)

input_ids = torch.tensor([tokenizer.encode(<span class="hljs-string">"Wikipedia was used to"</span>)])  <span class="hljs-comment"># batch size of 1</span>

language_id = tokenizer.lang2id[<span class="hljs-string">"en"</span>]  <span class="hljs-comment"># 0</span>
langs = torch.tensor([language_id] * input_ids.shape[<span class="hljs-number">1</span>])  <span class="hljs-comment"># torch.tensor([0, 0, 0, ..., 0])</span>

<span class="hljs-comment"># We reshape it to be of size (batch_size, sequence_length)</span>
langs = langs.view(<span class="hljs-number">1</span>, -<span class="hljs-number">1</span>)  <span class="hljs-comment"># is now of shape [1, sequence_length] (we have a batch size of 1)</span>

outputs = model(input_ids, langs=langs)
