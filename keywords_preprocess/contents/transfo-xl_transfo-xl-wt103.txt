<p><strong>Model Description:</strong>
The Transformer-XL model is a causal (uni-directional) transformer with relative positioning (sinuso√Ødal) embeddings which can reuse previously computed hidden-states to attend to longer context (memory). This model also uses adaptive softmax inputs and outputs (tied).</p>

<span>
		Misuse and Out-of-scope Use
	</span>
</h4>
<p>The model should not be used to intentionally create hostile or alienating environments for people. In addition, the model was not trained to be factual or true representations of people or events, and therefore using the model to generate such content is out-of-scope for the abilities of this model.</p>

<span>
		Training Data
	</span>
</h4>
<p>The authors provide additionally notes about the vocabulary used, in the <a href="https://arxiv.org/pdf/1901.02860.pdf" rel="nofollow">associated paper</a>: </p>
<blockquote>
<p>best model trained the Wikitext-103 dataset. We seed the our Transformer-XL with a context of at most 512 consecutive tokens randomly sampled from the test set of Wikitext-103. Then, we run Transformer-XL to generate a pre-defined number of tokens (500 or 1,000 in our case). For each generation step, we first find the top-40 probabilities of the next-step distribution and sample from top-40 tokens based on the re-normalized distribution. To help reading, we detokenize the context, the generated text and the reference text.</p>
</blockquote>
