<span>
		Model Description
	</span>
</h2>
<p>The XLM-RoBERTa model was proposed in <a href="https://arxiv.org/abs/1911.02116" rel="nofollow">Unsupervised Cross-lingual Representation Learning at Scale</a> by Alexis Conneau, Kartikay Khandelwal, Naman Goyal, Vishrav Chaudhary, Guillaume Wenzek, Francisco Guzm√°n, Edouard Grave, Myle Ott, Luke Zettlemoyer and Veselin Stoyanov. It is based on Facebook's RoBERTa model released in 2019. It is a large multi-lingual language model, trained on 2.5TB of filtered CommonCrawl data. This model is <a href="https://huggingface.co/xlm-roberta-large" rel="nofollow">XLM-RoBERTa-large</a> fine-tuned with the <a href="https://huggingface.co/datasets/conll2002" rel="nofollow">CoNLL-2002</a> dataset in Dutch.</p>
<ul>
<li><strong>Developed by:</strong> See <a href="https://arxiv.org/abs/1911.02116" rel="nofollow">associated paper</a></li>
<li><strong>Model type:</strong> Multi-lingual language model</li>
<li><strong>Language(s) (NLP):</strong> XLM-RoBERTa is a multilingual model trained on 100 different languages; see <a href="https://github.com/facebookresearch/fairseq/tree/main/examples/xlmr" rel="nofollow">GitHub Repo</a> for full list; model is fine-tuned on a dataset in Dutch</li>
<li><strong>License:</strong> More information needed</li>
<li><strong>Related Models:</strong> <a href="https://huggingface.co/roberta-base" rel="nofollow">RoBERTa</a>, <a href="https://huggingface.co/docs/transformers/model_doc/xlm" rel="nofollow">XLM</a><ul>
<li><strong>Parent Model:</strong> <a href="https://huggingface.co/xlm-roberta-large" rel="nofollow">XLM-RoBERTa-large</a></li>
</ul>
</li>
<li><strong>Resources for more information:</strong> 
-<a href="https://github.com/facebookresearch/fairseq/tree/main/examples/xlmr" rel="nofollow">GitHub Repo</a>

<span>
		Direct Use
	</span>
</h2>
<p>The model is a language model. The model can be used for token classification, a natural language understanding task in which a label is assigned to some tokens in a text. </p>
