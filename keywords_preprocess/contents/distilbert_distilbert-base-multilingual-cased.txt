<span>
		Model Description
	</span>
</h2>
<p>This model is a distilled version of the <a href="https://huggingface.co/bert-base-multilingual-cased/" rel="nofollow">BERT base multilingual model</a>. The code for the distillation process can be found <a href="https://github.com/huggingface/transformers/tree/main/examples/research_projects/distillation" rel="nofollow">here</a>. This model is cased: it does make a difference between english and English.</p>
<p>The model is trained on the concatenation of Wikipedia in 104 different languages listed <a href="https://github.com/google-research/bert/blob/master/multilingual.md#list-of-languages" rel="nofollow">here</a>.
The model has 6 layers, 768 dimension and 12 heads, totalizing 134M parameters (compared to 177M parameters for mBERT-base).
On average, this model, referred to as DistilmBERT, is twice as fast as mBERT-base.</p>
<p>We encourage potential users of this model to check out the <a href="https://huggingface.co/bert-base-multilingual-cased" rel="nofollow">BERT base multilingual model card</a> to learn more about usage, limitations and potential biases.</p>

<span>
		Direct Use and Downstream Use
	</span>
</h2>
<p>You can use the raw model for either masked language modeling or next sentence prediction, but it's mostly intended to be fine-tuned on a downstream task. See the <a href="https://huggingface.co/models?filter=bert" rel="nofollow">model hub</a> to look for fine-tuned versions on a task that interests you.</p>
<p>Note that this model is primarily aimed at being fine-tuned on tasks that use the whole sentence (potentially masked) to make decisions, such as sequence classification, token classification or question answering. For tasks such as text generation you should look at model like GPT2.</p>

<span>
		Out of Scope Use
	</span>
</h2>
<p>The model should not be used to intentionally create hostile or alienating environments for people. The model was not trained to be factual or true representations of people or events, and therefore using the models to generate such content is out-of-scope for the abilities of this model.</p>

<span>
		Bias, Risks, and Limitations
	</span>
</h1>
<p>Significant research has explored bias and fairness issues with language models (see, e.g., <a href="https://aclanthology.org/2021.acl-long.330.pdf" rel="nofollow">Sheng et al. (2021)</a> and <a href="https://dl.acm.org/doi/pdf/10.1145/3442188.3445922" rel="nofollow">Bender et al. (2021)</a>). Predictions generated by the model may include disturbing and harmful stereotypes across protected classes; identity characteristics; and sensitive, social, and occupational groups.</p>
