<span>
		Model Details
	</span>
</h2>
<p><strong>Model Description:</strong> This model is a fine-tune checkpoint of <a href="https://huggingface.co/distilbert-base-uncased" rel="nofollow">DistilBERT-base-uncased</a>, fine-tuned on SST-2.
This model reaches an accuracy of 91.3 on the dev set (for comparison, Bert bert-base-uncased version reaches an accuracy of 92.7).</p>

<span>
		How to Get Started With the Model
	</span>
</h2>
<p>Example of single-label classification:
​​</p>
<pre><code class="language-python"><span class="hljs-keyword">import</span> torch
<span class="hljs-keyword">from</span> transformers <span class="hljs-keyword">import</span> DistilBertTokenizer, DistilBertForSequenceClassification

tokenizer = DistilBertTokenizer.from_pretrained(<span class="hljs-string">"distilbert-base-uncased-finetuned-sst-2-english"</span>)
model = DistilBertForSequenceClassification.from_pretrained(<span class="hljs-string">"distilbert-base-uncased-finetuned-sst-2-english"</span>)

inputs = tokenizer(<span class="hljs-string">"Hello, my dog is cute"</span>, return_tensors=<span class="hljs-string">"pt"</span>)
<span class="hljs-keyword">with</span> torch.no_grad():
    logits = model(**inputs).logits

predicted_class_id = logits.argmax().item()
model.config.id2label[predicted_class_id]
</code></pre>

<span>
		Misuse and Out-of-scope Use
	</span>
</h4>
<p>The model should not be used to intentionally create hostile or alienating environments for people. In addition, the model was not trained to be factual or true representations of people or events, and therefore using the model to generate such content is out-of-scope for the abilities of this model.</p>

<span>
		Risks, Limitations and Biases
	</span>
</h2>
<p>Based on a few experimentations, we observed that this model could produce biased predictions that target underrepresented populations.</p>
<p>For instance, for sentences like <code>This film was filmed in COUNTRY</code>, this binary classification model will give radically different probabilities for the positive label depending on the country (0.89 if the country is France, but 0.08 if the country is Afghanistan) when nothing in the input indicates such a strong semantic shift. In this <a href="https://colab.research.google.com/gist/ageron/fb2f64fb145b4bc7c49efc97e5f114d3/biasmap.ipynb" rel="nofollow">colab</a>, <a href="https://twitter.com/aureliengeron" rel="nofollow">Aurélien Géron</a> made an interesting map plotting these probabilities for each country.</p>
<img alt="Map of positive probabilities per country." src="https://huggingface.co/distilbert-base-uncased-finetuned-sst-2-english/resolve/main/map.jpeg" width="500"/>
<p>We strongly advise users to thoroughly probe these aspects on their use-cases in order to evaluate the risks of this model. We recommend looking at the following bias evaluation datasets as a place to start: <a href="https://huggingface.co/datasets/wino_bias" rel="nofollow">WinoBias</a>, <a href="https://huggingface.co/datasets/super_glue" rel="nofollow">WinoGender</a>, <a href="https://huggingface.co/datasets/stereoset" rel="nofollow">Stereoset</a>.</p>
