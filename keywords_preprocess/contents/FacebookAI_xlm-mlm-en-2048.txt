<span>
		Model Details
	</span>
</h1>
<p>The XLM model was proposed in <a href="https://arxiv.org/abs/1901.07291" rel="nofollow">Cross-lingual Language Model Pretraining</a> by Guillaume Lample and Alexis Conneau. It’s a transformer pretrained with either a causal language modeling (CLM) objective (next token prediction), a masked language modeling (MLM) objective (BERT-like), or
a Translation Language Modeling (TLM) object (extension of BERT’s MLM to multiple language inputs). This model is trained with a masked language modeling objective on English text. </p>

<span>
		How to Get Started with the Model
	</span>
</h1>
<p>Use the code below to get started with the model. See the <a href="https://huggingface.co/docs/transformers/model_doc/xlm" rel="nofollow">Hugging Face XLM docs</a> for more examples.</p>
<pre><code class="language-python"><span class="hljs-keyword">from</span> transformers <span class="hljs-keyword">import</span> XLMTokenizer, XLMModel
<span class="hljs-keyword">import</span> torch

tokenizer = XLMTokenizer.from_pretrained(<span class="hljs-string">"xlm-mlm-en-2048"</span>)
model = XLMModel.from_pretrained(<span class="hljs-string">"xlm-mlm-en-2048"</span>)

inputs = tokenizer(<span class="hljs-string">"Hello, my dog is cute"</span>, return_tensors=<span class="hljs-string">"pt"</span>)
outputs = model(**inputs)

last_hidden_states = outputs.last_hidden_state
</code></pre>
